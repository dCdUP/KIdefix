Es wird die Webiste https://teachablemachine.withgoogle.com/ genutzt um ein Model zu erzeugen, welchen custom Bilder erkennt. Das Model kann selbst exportiert werden als Keras, dazu muss aber anscheinend auch Tensoflow installiert werden. Da Keras auf Python basiert, muss man testen ob man den Code nicht einfach nehmen kann und selbst ausführen kann.  
Man kann das Keras file mit ddem Python Code von "detecting custom objects" aufrufen und ausführen. Beide Dateien müssen in einem Ordner sein.
